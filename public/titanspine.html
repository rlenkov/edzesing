Titan's Spine

Training Description:

If the timer runs out or you fail the exercise you got 0 points/dmg
You can tick completed only if you completed the set 100% according to the given weight/reps. If you fell even 1 rep short or run out of time the set has failed.
Your score adds up based on completed sets and the difficulty chosen for each set
To complete the challenge you need at least 1 brutal and 1 hard set completed for each exercise on average.
